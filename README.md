# Percorsi Occitani Water Overlay

Ein Offline-Kartenprojekt für die Percorsi Occitani.

## Ziel

Erzeugung von GPX- und KML-Overlays für:

- Guru Maps

- MapOut

- Topo GPS

## Enthaltene Ebenen

- 💧 Trinkwasser

- ⛲ Brunnen

- 💦 Quellen

- 🍞 Versorgung

- ⚠️ Kritische Wasserabschnitte

## Datenquellen

- Offizielle GPX-Tracks der Percorsi Occitani

- OpenStreetMap

- Wanderberichte (optional)
